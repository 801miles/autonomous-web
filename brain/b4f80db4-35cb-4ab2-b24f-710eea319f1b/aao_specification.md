# AAO Specification: Intake Matrix & Prompt Transmutation

**Objective**: To establish a high-fidelity information extraction layer that leverages psychology and engineering to guarantee autonomous build alignment.

---

## 1. The "Golden Questions" (Intake Matrix)

The following questions are designed to minimize cognitive load while maximizing information density.

1. **Utility Synthesis**: "What is the single non-negotiable conversion or outcome this platform must achieve in its first 30 seconds of interaction?"
2. **Psychographic Mapping**: "Who is the 'End-User' today? Describe their dominant emotional friction (e.g., overwhelmed, skeptical, urgent) and how your solution resolves it."
3. **Identity Anchors**: "Name three values (e.g., 'Radical Transparency', 'Cold Efficiency', 'Wild Creativity') that must influence every design and code choice."
4. **Trust Architecture**: "What specific social proof or data proof exists that will make a skeptical visitor feel safe?"
5. **Agent Synchronization**: "Which Knowledge Items or Agent Contexts should be 'Observed' to align this build with your existing AI profiles?"

---

## 2. The Backend Prompt Template (Transmutation)

This template is used by the AAO Meta-Brain to expand the simple answers above into a 2,000-word Technical Specification for sub-agents (Dev, Designer, DevOps).

```markdown
### [UNIT_ORCHESTRATION_DIRECTIVE]
**Role**: Antigravity Architect Orchestrator (AAO)
**Units**: [Engineer, UX Lead, ToM Specialist, Social Psychologist]
**Framework**: Next.js 15, React 19, Tailwind v4

#### 1. INTENT EXPANSION (Deconstruction of User Input)
- **Parse Input {Utility}**: Map to a conversion-focused landing page organism.
- **Parse Input {Psychography}**: Calibrate UX friction levels. If 'Skeptical', increase data-density and trust-marks.
- **Parse Input {Identity}**: Generate a curated CSS palette and typography scale reflecting brand values.
- **Parse Input {Proof}**: Strategic placement of 'Organisms' (Social Proof sections).

#### 2. ATOMIC DESIGN SPECIFICATION
- **Atoms**: Functional primitives (Buttons, Inputs, Spacing tokens).
- **Molecules**: Combined primitives (Search bars, Form fields, Hero CTAs).
- **Organisms**: Complex UI sections (Navigation, Infinite Scroll Galleries, Feature Grids).
- **Templates**: Page structures (App Router Layouts).

#### 3. SUB-AGENT DIRECTIVES
- **To Dev Agent**: "Implement Server Components for data-fetching and Client Components for {Psychography}-driven interactions. Ensure O(1) performance."
- **To Designer Agent**: "Craft a UI that addresses the user's {Identity Anchor}. Apply rich aesthetics (Midnight Luxe) with tailored micro-animations."
- **To DevOps Agent**: "Prepare the repository for immediate GitHub initialization and production-grade hosting."

#### 4. ASSET SYNTHESIS
- **Query**: "Generate {exact_query_of_images} representing {Utility}."
- **Wireframe**: "Synthesize SVG wireframe JSON based on Atomic Page Template."
```
